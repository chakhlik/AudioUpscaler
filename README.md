# Audio Upscaler

## Назначение
Утилита командной строки для повышения частоты дискретизации и разрядности аудиофайлов (апскейлинг).

## Технические характеристики
*   **Входной формат:** WAV, 44.1 кГц, 16 бит, стерео (CD-Audio).
*   **Выходной формат:** WAV, 176.4 кГц, 24 бит, стерео (HiRes-Audio).
*   **Обработка:** 4-кратная передискретизация.

## Методы интерполяции
| Метод | Описание |
| :--- | :--- |
| `cubic` | Кубическая интерполяция (по умолчанию). |
| `akima` | Интерполяция Акима. |
| `pchip` | Кубическая интерполяция с сохранением формы. |
| `repeat` | Без интерполяции (повторение отсчетов). |

## Параметры командной строки
```text
python upscaler.py <input_file> <output_file> [--method <method>]
```

| Аргумент | Статус | По умолчанию | Описание |
| :--- | :---: | :---: | :--- |
| **input_file** | Обязательный | — | Путь к исходному WAV файлу. |
| **output_file** | Обязательный | — | Путь и имя выходного файла. |
| **--method** | Необязательный | `cubic` | Выбор алгоритма интерполяции. |

## Примеры использования
```bash
# Базовый запуск (метод cubic)
python upscaler.py track.wav track_hires.wav

# Выбор метода интерполяции
python upscaler.py track.wav track_hires.wav --method akima

# Без интерполяции
python upscaler.py track.wav track_hires.wav --method repeat
```

## Результат
Выходной файл создается по указанному пути (рекомендуется та же папка, что у исходного файла).
